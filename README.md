# Tahu Search

## Apa Itu Tahu Search?
Tahu Search adalah website search engine sederhana yang dibuat untuk menyelesaikan tugas akhir mata kuliah Temu Kembali Informasi.
Link Website Tahu Search: https://search-engine-backend-xi.vercel.app/
